onBeat

OnBeat is a web application designed to help users make playlists based on song tempo.

  • User authentication written from scratch using Bcrypt-Ruby gem to secure passwords.
  • Utilized Test-Driven-Development practices by writing a full test suite using rspec.

docket

Docket is a single page application for users to find events and schedule them into a calendar.

Interesting Details:

  • Built nested routing into the page using AngularUI-Router, which allowed me to organize the site around nested states.
  • Used UI-Calendar wrapper for the jQuery-based Full Calendar to display events saved by users into a monthly and weekly formatted calendar.
  • Utilized Eventful API calls for users to search events based on date and location.

    • Built Using:
